Fundamentos da Tecnologia da Informação

O que é Hardware

Hardware envolve qualquer componente físico que compõe um aparelho eletrônico. Os hardwares costumam ser relacionados somente aos computadores, mas também abrangem outros eletrônicos da tecnologia de consumo.


O que é uma placa mãe?

É a placa de circuito principal de um computador que conecta e permite a comunicação entre todos os componentes de hardware, como processador, memória RAM, disco rígido e placa de vídeo.


O que é um processador?

É o "cérebro" de um computador ou dispositivo eletrônico, sendo o componente central responsável por executar instruções e realizar cálculos para processar dados.


O que é memória RAM

É o armazenamento de acesso rápido em um computador que guarda os dados e instruções que estão sendo usados ativamente pelo processador (CPU)


O que é um Sistema Operacional?

Um sistema operacional (SO) é o conjunto de programas que gerencia os recursos de hardware e software de um computador, permitindo que o usuário interaja com a máquina e que outros programas (como aplicativos) possam ser executados.


  • Windows - desenvolvido pela Microsoft, muito usado em computadores pessoais.
  • macOS – sistema operacional dos computadores Apple (Mac).
  • Linux – sistema de código aberto usado em servidores e também em desktops.
  • Android - sistema operacional baseado em Linux, usado em smartphones e tablets.

O que é um sistema de arquivos?

O sistema de arquivos é o método que o sistema operacional usa para armazenar, organizar e gerenciar os dados em um dispositivo de armazenamento — como o HD, SSD, pendrive ou cartão de memória.


Ele define como os arquivos são nomeados, gravados, acessados e estruturados em pastas (ou diretórios).
